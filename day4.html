COMPARE COMPARISON OPERATOR AND LOGICAL OPERATOR

Purpose:

Comparison Operators Logical Operators
Used to compare two values or expressions to determine the relationship between them. Used to combine multiple conditions or to negate a condition.
Examples: `==` (equal to), `!=` (not equal to), `>` (greater than), `<` (less than), `>=` (greater than or equal to), `<=` (less than or equal to). Examples: `&&` (logical AND), `||` (logical OR), `!` (logical NOT).
They produce a boolean result (`true` or `false`) based on the comparison. They operate on boolean values (`true` or `false`) and produce boolean results.

Operands:

Comparison Operators Logical Operators
Operate on operands that are typically values or expressions whose values can be compared. Operate on boolean operands (`true` or `false` values) derived from comparison operations or other boolean expressions.
Example: `x > y` compares the values of `x` and `y`. Example: `(x > 0) && (y < 10)` combines the results of two comparison operations using logical AND.

Output:

Comparison Operators Logical Operators
Return a boolean (`true` or `false`) based on whether the comparison condition is satisfied. Combine boolean values or negate them to produce a boolean result.
Example: `5 > 3` returns `true`. Example: `(5 > 3) && (10 < 5)` returns `false` because the second condition is not true.

Usage:

Comparison Operators Logical Operators
Used primarily to evaluate conditions based on values. Used to create complex conditions by combining multiple simpler conditions.
Example: Checking if a number is within a certain range. Example: Checking if a number is both positive and less than 100 (`(x > 0) && (x < 100)`).
