JavaScript Loop Examples

For Loop

When to Use: Use a for loop when you know the number of iterations ahead of time or when you need to iterate over a sequence or range of values. Ideal for counting iterations.

<script>
  for (let i = 1; i <= 5; i++) {
    console.log(i);
  }
</script>

While Loop

When to Use: Use a while loop when the number of iterations is not known beforehand, and you want to repeat a block of code as long as a condition is true. Useful when continuation depends on runtime conditions.

<script>
  let number;
  while (number <= 10 || isNaN(number)) {
    number = Number(prompt("Enter a number greater than 10:"));
  }
  console.log("Valid number entered:", number);
</script>

Do-While Loop

When to Use: Use a do-while loop when you want to ensure that the loop executes at least once before checking the condition. Useful for scenarios where the loop body must run at least one time.

<script>
  let continueLoop;
  do {
    continueLoop = confirm("Do you want to continue?");
  } while (continueLoop);
  console.log("Loop ended");
</script>
